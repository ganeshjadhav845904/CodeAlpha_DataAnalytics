# 📊 Titanic Dataset Dashboard

## 📌 Project Overview

This project was developed as part of the **CodeAlpha Data Analytics Internship**.

The objective of this project is to create an interactive dashboard that presents important insights from the Titanic dataset using Python visualization libraries.

---

# 🎯 Objective

To transform raw data into meaningful visualizations that help understand passenger demographics and survival patterns.

---

# 🛠 Technologies Used

- Python
- Pandas
- Matplotlib

---

# 📈 Dashboard Visualizations

The dashboard contains the following visualizations:

- Passenger Survival Count
- Gender Distribution
- Passenger Class Distribution
- Age Distribution

---

# 📊 Key Insights

- Most passengers did not survive the accident.
- Male passengers were more than female passengers.
- Most passengers traveled in Passenger Class 3.
- The majority of passengers were between 20 and 35 years old.

---

# 📁 Output

The project generates:

- `dashboard.png`

which contains all important visualizations in a single dashboard.

---

# ✅ Conclusion

The dashboard successfully summarizes important information from the Titanic dataset in a clean and easy-to-understand visual format. It helps identify passenger distribution, survival trends, and demographic patterns through effective data visualization.
