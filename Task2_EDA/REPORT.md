# 📊 Exploratory Data Analysis (EDA) - Titanic Dataset

## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, and insights using Python.

---

# 🎯 Objective

To clean the dataset, analyze important features, and visualize the data to better understand passenger demographics and survival patterns.

---

# 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

# 📂 Dataset

**Dataset:** Titanic Dataset

The dataset contains passenger information such as age, gender, ticket class, fare, and survival status.

---

# ⚙️ Steps Performed

- Loaded the dataset
- Explored dataset structure
- Handled missing values
- Removed unnecessary columns
- Performed descriptive statistics
- Generated visualizations
- Identified relationships between variables

---

# 📊 Visualizations

- Passenger Survival Count
- Gender Distribution
- Passenger Class Distribution
- Age Distribution
- Fare Distribution
- Correlation Heatmap

---

# 📈 Key Insights

- Most passengers did not survive the disaster.
- Passenger Class 3 had the highest number of passengers.
- Male passengers were more than female passengers.
- Most passengers were between 20–35 years old.
- Fare distribution was highly skewed with a few high-value tickets.
