# 😊 Sentiment Analysis using TextBlob

## 📌 Project Overview

This project was developed as part of the **CodeAlpha Data Analytics Internship**.

The objective of this project is to analyze customer reviews and classify them into **Positive**, **Negative**, and **Neutral** sentiments using the **TextBlob** Natural Language Processing (NLP) library.

---

# 🎯 Objective

To perform sentiment analysis on customer reviews by calculating sentiment polarity and categorizing each review based on its emotional tone.

---

# 🛠 Technologies Used

- Python
- TextBlob
- Natural Language Processing (NLP)

---

# 📂 Dataset

The project uses a collection of sample customer reviews for sentiment classification.

---

# ⚙️ Workflow

1. Import the TextBlob library.
2. Load customer reviews.
3. Calculate sentiment polarity for each review.
4. Classify reviews into:
   - 😊 Positive
   - 😞 Negative
   - 😐 Neutral
5. Display sentiment results and summary.

---

# 📊 Output

The program generates the following information for every review:

- Review Text
- Polarity Score
- Sentiment Category

It also displays the total number of:

- Positive Reviews
- Negative Reviews
- Neutral Reviews

---

# 📈 Learning Outcomes

- Understanding of Natural Language Processing (NLP)
- Text preprocessing concepts
- Sentiment polarity calculation
- Opinion mining using Python

---

# ✅ Conclusion

The project successfully demonstrates sentiment analysis using the TextBlob library. It classifies customer reviews based on polarity scores and provides a simple yet effective example of Natural Language Processing for opinion mining.
